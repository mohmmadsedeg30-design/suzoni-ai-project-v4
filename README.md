# Suzoni AI

Custom AI with built-in reasoning. Arabic + English support.

## Quick Start

```bash
python main.py setup
python main.py train --data data.txt
python main.py chat
```

## Integration

```bash
python main.py integrate --model gpt2
```

## Files

- config.py - Settings
- tokenizer.py - BPE Tokenizer
- model.py - Neural Network
- trainer.py - Training
- integrator.py - External models
- chat.py - Interface
- main.py - Entry point
